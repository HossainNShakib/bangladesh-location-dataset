🎯 What This Is:
   A clean, reusable, developer-friendly dataset containing Bangladesh's 
   administrative divisions with support for multiple formats.

🌳 Data Hierarchy:
   Division (1)
     ├─ District (4)
     │   ├─ SubDistrict (12)
     │   │   └─ LocalArea (6)

📋 Included Formats:
   ✓ JSON (flat & nested)
   ✓ CSV (spreadsheet-ready)
   ✓ SQL (PostgreSQL & MySQL)
   ✓ JavaScript (ES Modules)
