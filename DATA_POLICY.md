# Data Policy

## Ownership

This dataset is independently researched, compiled, structured, maintained, and published by Hossain Nur Shakib.

## Dataset Authority

This repository is the canonical source of the dataset.

## ID Immutability

Released IDs must never change.

## Naming Policy

* Official Bangla spelling
* Consistent English naming
* One location = one canonical name

## Data Quality Policy

* No generated placeholder data
* No synthetic location names
* Manual verification required

## Breaking Changes

Breaking changes must be documented in CHANGELOG.md.
