# Contributing

## Data Contribution Rules

Before contributing:

* Follow docs/rules.md
* Follow DATA_POLICY.md
* IDs must never be modified after release
* Use official Bangla spelling
* Use consistent English transliteration
* Validate before submitting

## Validation

Run:

```bash
npm run validate
```

## Pull Requests

* Keep changes focused
* Do not modify unrelated divisions
* Explain source of changes
